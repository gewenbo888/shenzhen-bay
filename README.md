# Shenzhen Bay Protocol · 深圳湾行动

A fictional cinematic AAA-game landing concept set in a near-future cyberpunk Shenzhen Bay. Six operatives, six districts, six missions — drones, plasma rifles, EMP-locked civic core, maglev underline. Bilingual EN + 中文 throughout.

> **This is not a real game and not a real product.** It is a stylized concept landing page, designed to demonstrate cinematic web UI for a fictional sci-fi tactical action title.

## Links

- **Live:** [shenzhen-bay.psyverse.fun](https://shenzhen-bay.psyverse.fun)
- **GitHub:** [github.com/gewenbo888/shenzhen-bay](https://github.com/gewenbo888/shenzhen-bay)

## Stack

- Next.js 14 (App Router, `src/app/`)
- React 18 + TypeScript
- Canvas 2D — parallax skyline with rain, drones, neon billboards, fog, bay reflection
- Three.js — rotating holographic wireframe tower (briefing panel)
- SVG — tactical district map, drone-feed minimap, operative portraits
- Framer Motion — scroll reveal
- Tailwind CSS with a custom cyberpunk palette (cyan + magenta + amber on near-black)
- Fonts: Orbitron display, Rajdhani body, JetBrains Mono HUD, ZCOOL XiaoWei + Noto Sans SC
